Penguin-X Project

POW Scrypt algorithm cryptocurrency

Forked from Litecoin Core version 0.15, the Penguin-X project is a fully functional digital currency. Created for educational and experiential purposes, with the intention of retaining all the current functionality of Litecoin. Penguin-X operates on a freshly generated blockchain retaining many of the original design characteristics.

Specifications

  • Proof of Work   : "Mining" drives network and creates currency value
  • Market Symbol : "PNGX"
  • Block Reward   : 50 PNGX
  • Block Time        : ~10 minutes per block
  • Supply Limit     : 84,000,000 PNGX
